# Task Manager — Frontend

This is a Vite + React + TypeScript UI for the Task Manager.

Quick start (Windows PowerShell):

1. Install dependencies

```powershell
cd "e:\MY PROJECTS\Task Manager\frontend"
npm install
```

2. Run dev server

```powershell
npm run dev
```

3. Build

```powershell
npm run build
npm run preview
```

Notes:
- The frontend expects the backend at http://localhost:4000 by default. To change the backend base URL, add a `.env` file with `VITE_API_BASE=http://your-api`.
